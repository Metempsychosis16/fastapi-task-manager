# FastAPI Task Manager API

Modern backend API built using FastAPI, PostgreSQL, JWT Authentication, and SQLAlchemy.

## Features

- JWT Authentication
- User Registration/Login
- CRUD Task Management
- PostgreSQL Integration
- File Upload Support
- Pagination/Search/Filtering
- Protected Routes
- Docker Support
- Pytest Testing

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT
- Docker
- Pytest

## Run Locally

pip install -r requirements.txt

uvicorn app.main:app --reload
